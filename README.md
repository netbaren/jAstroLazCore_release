# jAstroLazCore

jAstroLazCore is a desktop astronomy application for exploring sky phenomena,
time systems, seasons, solar diagrams, ephemerides and observation views.

This public repository is used for binary releases only. The private
Pascal/Lazarus source tree is not included here.

## Downloads

Prebuilt packages are published from the GitHub **Releases** page.

For Windows, download the `jAstroLaz-win64-*.zip` archive, extract it to a
writable folder, then run:

```text
jastro_lazcore_app.exe
```

Keep the `resources` folder next to the executable.

For a step-by-step guide, see [INSTALLATION_WINDOWS.md](INSTALLATION_WINDOWS.md).

## Platforms

- Windows 64-bit: available first.
- Linux x86_64: planned, to be provided as a separate package after testing.
